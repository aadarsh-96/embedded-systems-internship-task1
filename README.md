# Motion Detection Alarm using Arduino UNO

## Overview

This project demonstrates a Motion Detection Alarm using an Arduino UNO and a PIR Motion Sensor. The system detects human movement and activates an LED (or buzzer) as an alert.

---

## Objectives

- Learn the basics of Embedded Systems.
- Interface a PIR Motion Sensor with Arduino UNO.
- Detect human motion.
- Generate an alert using LED/Buzzer.

---

## Components Used

- Arduino UNO
- PIR Motion Sensor
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

## Working Principle

1. PIR sensor detects human movement.
2. Sensor sends HIGH signal to Arduino.
3. Arduino processes the signal.
4. LED turns ON.
5. When no motion is detected, LED turns OFF.
